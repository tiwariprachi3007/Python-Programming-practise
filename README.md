# 🐍 Python Programming Practice

A structured repository to practice **each and every Python concept** with organized folders, clean scripts, and recruiter‑friendly presentation.

---

## 📂 Folder Structure

### `variable_and_data_types/`
This folder contains beginner‑friendly programs to understand variables and data types in Python.

- **first_program.py** → Prints **Hello World** and demonstrates combining two lines of output into a single print statement.  
- **exp_execution.py** → Demonstrates expression execution and evaluation in Python.  
- **sum.py** → Program to calculate the sum of two numbers using variables.  
- **difference.py** → Program to calculate the difference between two numbers using variables.
